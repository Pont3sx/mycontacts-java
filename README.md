# 📒 MyContacts — Agenda de Contatos em Java

## 📌 Sobre o projeto

O **MyContacts** é uma aplicação em Java desenvolvida para gerenciar contatos através de um menu interativo no terminal.
O sistema permite adicionar, listar, buscar por nome e remover contatos, aplicando conceitos fundamentais de programação orientada a objetos.

Este projeto foi criado com foco em aprendizado e prática de boas práticas de desenvolvimento, reforçando o uso de POO, manipulação de lista e pesquisa de dados simples.

---

## 🚀 Funcionalidades

* ✅ Adicionar novos contatos
* ✅ Listar contatos cadastrados
* ✅ Buscar contato por nome
* ✅ Remover contato
* ✅ Tratamento de exceções personalizadas
* ✅ Estrutura organizada em camadas (Model, Controller, App)

---

## 🛠 Tecnologias utilizadas

* Java
* Programação Orientada a Objetos (POO)
* ArrayList
* Tratamento de Exceções
* IntelliJ IDEA

## ▶️ Como executar

1. Clone o repositório
2. Abra no IntelliJ ou outra IDE Java
3. Execute a classe `Main`
4. Use o menu interativo no terminal
